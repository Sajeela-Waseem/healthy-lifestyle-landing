# Eco-Friendly Kitchenware Landing Page

Theme: **Eco-Friendly Kitchenware for a Greener Home**

This React + Vite + Tailwind landing page displays curated kitchen products and links each item to an Amazon affiliate URL.

## Getting started

1. Install dependencies

```bash
cd landing-page
npm install
```

2. Run dev server

```bash
npm run dev
```

## Product data and theme reference

- Product names in `src/App.jsx`
  - `Bamboo Utensil Set`
  - `Ceramic Cookware Set`
  - `Reusable Glass Storage`
- Product link label in `src/components/ProductCard.jsx`
  - `Buy on Amazon`
- Hero section buttons
  - `Shop Best Sellers` → `#products`
  - `Explore the Collection` → `#about`
- Theme reference in this document: `Eco-Friendly Kitchenware for a Greener Home`

## Notes

- Replace the placeholder Amazon affiliate URLs in `src/App.jsx` with your own Amazon Associate links.
- Replace product image URLs with your own images or local assets if preferred.
